# Environment variables

## apps/theme

```bash
SHOPIFY_FLAG_STORE=your-store.myshopify.com
SHOPIFY_CLI_THEME_TOKEN=***
SHOPIFY_THEME_ID=123456789
```

## apps/admin

```bash
SHOPIFY_API_KEY=***
SHOPIFY_API_SECRET=***
SHOPIFY_ADMIN_SCOPES=read_products,read_inventory,write_products
HOST=https://your-tunnel.ngrok.io
```

Add these to `.env` files (not committed). Use `.env.example` as a template.
